# Privacy Policy for SleepTags

**Last Updated:** December 2025

## Overview

SleepTags is a browser extension that helps optimize RAM usage by discarding inactive tabs. This policy explains how we handle your data.

## Data Collection

- **Tab Data:** We access tab information (URLs, titles, active state) locally within your browser to identify inactive tabs eligible for discarding.
- **System Memory:** We access system memory information locally to display available RAM in the extension popup.

## Data Usage

All collected data is used exclusively for the extension's core functionality:
- Identifying and freezing inactive browser tabs
- Displaying memory optimization statistics

## Data Storage

- User preferences and usage statistics are stored locally using Chrome's Local Storage API.
- No data is transmitted to external servers.

## Third-Party Sharing

- We do not sell, trade, or transfer any user data to third parties.
- No data is shared for marketing or advertising purposes.

## Data Security

All data processing occurs locally within your browser. No data leaves your device.
